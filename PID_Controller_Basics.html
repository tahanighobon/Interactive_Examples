What students should notice

  • P control responds immediately, but it may leave a steady-state error.
  • I control accumulates past error and helps remove offset.
  • D control reacts to how fast the error changes and can reduce overshoot.
  • Too much gain can make the system oscillate or become unstable.

A useful activity is to start with P only, then add I, then add D, and compare how the response changes each time.
