What students should notice

  • Each logic gate follows a fixed rule no matter how many times the inputs change.
  • The truth table lists every possible input combination and the corresponding output.
  • NOT is different because it uses only one input and inverts it.

Teaching idea

Ask students to predict the output before they press the input buttons. Then compare their prediction with the highlighted row in the truth table.
