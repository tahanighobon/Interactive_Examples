How to use

  • Change slope, intercept, and noise, then generate data.
  • Drag points on the graph to manually change the dataset.
  • The green line is the best-fit regression line.
  • The orange vertical segments show prediction errors.

What students should notice

  • The best-fit line minimizes the average squared prediction error.
  • More noise usually increases the MSE and makes the fitted line less exact.
  • Dragging one point far away can noticeably change the regression line.
  • Residuals show how far each point is from the model prediction.
