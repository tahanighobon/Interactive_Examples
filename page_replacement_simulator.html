What students should notice

  • FIFO removes the oldest loaded page, even if it may be used again soon.
  • LRU tries to keep recently used pages in memory.
  • Optimal gives the minimum possible faults because it replaces the page used farthest in the future.
  • The same reference string can behave very differently depending on the number of frames.
